\# VPS Technologies Enterprise Home Lab



This repository documents the design, deployment, and operation of a simulated enterprise infrastructure for VPS Technologies.



\## Current Status



\- Proxmox VE installed

\- TrueNAS configured

\- DC01 deployed and promoted to the first domain controller

\- Active Directory forest created: `corp.vpstechnologies.internal`

\- NetBIOS name: `VPSTECH`



\## Project Goals



\- Learn enterprise systems administration

\- Build and document an Active Directory environment

\- Practice Windows Server, Linux, storage, backup, and disaster recovery

\- Track the project using Git and GitHub
